# 🍽️ Web UMKM Kuliner Sederhana

Website profil UMKM kuliner sederhana yang dibangun sebagai project **UTS Praktikum Pemrograman Website**.

---

## 📋 Deskripsi

Website ini merupakan langkah atau bentuk digitalisasi dari sebuah UMKM di bidang kuliner. Menampilkan informasi usaha, daftar menu makanan, makanan khas, form pemesanan, hingga halaman kontak.
Semuanya dalam tampilan yang ringan dan mudah diakses didevice manapun.

---

## 🗂️ Struktur Halaman Website

| Halaman | File | Deskripsi |
|---|---|---|
| Beranda | `index.html` | Halaman utama / landing page |
| Menu | `menu.html` | Daftar menu makanan & minuman |
| Makanan Khas | `makanan-khas.html` | Informasi makanan khas unggulan |
| Pemesanan | `pemesanan.html` | Form pemesanan online |
| Profil | `profil.html` | Profil & informasi UMKM |
| Kontak | `kontak.html` | Informasi kontak & lokasi |

---

## 📁 Struktur Direktori

```
Web-UMKM-Kuliner-Sederhana/

```

---

## 🛠️ Teknologi yang Digunakan

- **HTML 5** — Struktur halaman website
- **CSS 3 (Vanila CSS)** — Styling dan layout website
- **JavaScript** — Interaktivitas halaman dan logic backend website

---

## 🚀 Cara Menjalankan

1. Clone repository ini:
   ```bash
   git clone https://github.com/Davin164/Web-UMKM-Kuliner-Sederhana.git
   ```
2. Buka folder project
3. Jalankan `index.html` di browser (bisa langsung double-click atau pakai Live Server di VS Code)
